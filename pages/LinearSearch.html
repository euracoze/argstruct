Correctness

We shall prove the correctness via induction. Let \(P(k)\) be the following proposition.

"Element \(1\) through \(k-1\) does not contains the key."

For the base case, when \(k=0\), element \(1\) through \(-1\) does not contains the key. Since that is an empty array, the invariant holds.

For the inductive case, at the beginning of \($k$th\) iteration, element \(1\) through \(k-1\) does not contains the key. We compare the \($k$th\) element with the key. If it matches, we return the index and the search terminates with the correct value. If it doesn't, we increment \(k\) by \(1\) to maintain the loop invariant, ensuring element \(1\) through \(k\) does not contains the key.

For the termination case, when \(k=n\), element \(1\) through \((n+1)-1\) does not contains the key. Since \((n+1)-1=n\), we get element \(1\) through \(n\) does not contains the key, and we return value to indicate element not found.
